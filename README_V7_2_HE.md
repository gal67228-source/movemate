# Gal Family Trips – V7.2

## יעדים
- רשימת היעדים הורחבה לערים רבות עם קווים ישירים או עונתיים מישראל.
- הרשימה כוללת אירופה, ארצות הברית, אסיה, אפריקה והמזרח התיכון.
- קיים חיפוש לפי עיר או מדינה.
- תמיד ניתן להוסיף יעד חופשי שאינו ברשימה.

## תאריכים לכל יעד
- כל יעד מקבל תאריך התחלה ותאריך סיום משלו.
- טווחי היעדים חייבים להיות רציפים וללא חפיפה.
- כל יום משויך ליעד אחד בלבד.
- מוצג מראש מספר הימים שייוצר.

## יצירת ימים אוטומטית
- בעת יצירת טיול נוצרים כל הימים לפי טווחי היעדים.
- כל יום נשמר עם שדה destination מפורש.
- כותרת ברירת המחדל היא "יום ב<עיר>".
- מזג האוויר משתמש קודם ביעד המפורש של היום.

## עריכת טיול
- נוסף כפתור עריכה לכל טיול.
- שינוי שם, יעדים או תאריכים בונה מחדש את רשימת הימים.
- פעילויות של תאריכים שנשארו נשמרות.
- ימים חדשים נוצרים ריקים.
- ימים שהוסרו מהטווח מוסרים.
- מסעדות ששויכו ליום שנמחק מוסרות מהשיוך.

## מסך ימים
- הוסר כפתור הוספת יום.
- הוסרה מחיקת יום.
- ניתן לערוך רק את כותרת היום.
- שינוי תאריך או יעד נעשה רק דרך עריכת פרטי הטיול.
