## מסך טיסות
לכל טיסה מזינים:
- מספר טיסה
- תאריך ושעת המראה
- תאריך ושעת נחיתה
- שדה יציאה ונחיתה
- מקום איסוף להסעה
- זמן הסעה לשדה
- זמן משוער לנחיתה ואיסוף מזוודות

המסלול נבנה אוטומטית:
- יציאה להסעה
- הגעה לשדה 3 שעות לפני ההמראה
- הטיסה
- נחיתה ואיסוף מזוודות

עריכה או מחיקה של טיסה בונה מחדש את השלד ומונעת כפילויות.

## מסך מלונות
בסיסי אירוח:
- לינה בלבד
- ארוחת בוקר
- חצי פנסיון
- פנסיון מלא

שעות קבועות:
- ארוחת בוקר 08:00
- ארוחת צהריים 13:00
- ארוחת ערב 19:00

כללי שהייה:
- ארוחת בוקר מתחילה בבוקר שאחרי הצ׳ק־אין וכוללת את יום הצ׳ק־אאוט.
- ארוחת ערב מתווספת ביום הצ׳ק־אין ועד הערב שלפני הצ׳ק־אאוט.
- ארוחת צהריים בפנסיון מלא מתווספת עד היום שלפני הצ׳ק־אאוט.

עריכה, שינוי בסיס אירוח או מחיקת מלון מעדכנים את כל הארוחות האוטומטיות.

## מניעת כפילויות
כל פעילויות השלד מקבלות מזהים יציבים. בכל שינוי:
1. הפעילויות האוטומטיות הישנות מוסרות.
2. השלד נבנה מחדש מהטיסות והמלונות.
3. פעילויות ידניות נשמרות.
