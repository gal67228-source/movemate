# Gal Family Trips – V7.8 Smart Timeline Editor

## הוספת פעילות בתוך ציר הזמן
- כפתור ההוספה הוסר מראש מסך היום.
- נוסף כפתור הוספה בתחילת היום.
- נוסף כפתור קטן בין כל שתי פעילויות.
- נוסף כפתור הוספה בולט בסוף היום.
- בכל כרטיס קיים כפתור + להוספה מיד אחרי הפעילות.

## חישוב שעות
- שעת פעילות חדשה נקבעת לפי סיום הפעילות הקודמת.
- הפעילויות הגמישות שאחריה מחושבות מחדש.
- שינוי משך או גרירת פעילות מחשבים מחדש את המשך היום.
- בכל כרטיס מוצג טווח זמן, לדוגמה 09:00–11:00.

## שעה קבועה וגמישה
- ניתן לסמן פעילות כגמישה או קבועה.
- טיסות, צ׳ק-אין וצ׳ק-אאוט מזוהים אוטומטית כקבועים.
- פעילות קבועה שומרת את השעה המקורית.
- במקרה של חפיפה מוצגת אזהרה בראש היום.

## עיצוב
- כפתורי ההוספה משולבים בציר הזמן.
- מוצג סימון 🔒 קבועה או ↻ גמישה.
- כפתור + קטן נמצא בתוך כל כרטיס.
