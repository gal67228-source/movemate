# Gal Family Trips – V8.4.0

## מרכז מסמכים חכם

- קיבוץ מסמכים לפי טיסות, מלונות, אטרקציות,
  הסעות, תחבורה, ביטוח ומסמכים אישיים.
- דרישות נוצרות אוטומטית גם מרשימת הטיסות,
  ולא רק מפעילויות במסלול.
- לכל דרישה מוצג מצב: חסר, חלקי או מלא.
- סינון לפי קטגוריה.
- בטיסות ובדרכונים ניתן לצרף כמה מסמכים לכל נוסע.
- המסמכים מוצגים בקבוצות לפי שם הנוסע.
- לכל מסמך מוצג סוג הקובץ ומצב זמינות אופליין.
- נשמרים מזהה ההזמנה, תפקיד המסמך ותאריך ההוספה.
- פתיחה ומחיקה זמינות ישירות מתוך הכרטיס.
- מסמכים כלליים נשארו זמינים.
