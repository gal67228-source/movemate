# Gal Family Trips – V8.1.0.1

## כללי השלמת פעילות

- אי אפשר לסמן פעילות כהושלמה לפני שתאריך היום שלה הגיע.
- ביום עתידי כפתור ההשלמה מושבת ומוצגת הסיבה.
- פעילות שסומנה בטעות ניתנת לאיפוס כל עוד זמן הסיום המתוכנן שלה עדיין לא עבר.
- ביום עתידי ניתן לאפס נתון השלמה קיים.
- לאחר שיום הפעילות עבר, אי אפשר לאפס השלמה ממסך "היום".
- בכרטיסי ציר היום מופיע כפתור "איפוס" רק כשמותר לבצע את הפעולה.
