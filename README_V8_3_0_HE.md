# Gal Family Trips – V8.3.0

## Smart Home Dashboard

מסך הבית הוחלף בלוח בקרה חכם לטיול הפעיל.

### סיכום הטיול
- שם ויעד
- תאריכים
- ספירה לאחור / היום הנוכחי / טיול שהסתיים
- פס התקדמות

### כרטיסי מצב
- טיסה הבאה
- מלון פעיל או קרוב
- סכום הוצאות
- מסמכים חסרים

### הפעולה הבאה
הכרטיס מציג אוטומטית:
- צא לדרך
- אשר הגעה
- סיים פעילות
- השלמת מסמכים
- השלמת מלונות
- הטיסה הבאה

### מזג אוויר
מזג האוויר מוצג לפי היום והיעד הפעילים.

### Trip Health
הציון מתחשב ב:
- מסמכי טיסות ומלונות
- כיסוי לילות במלונות
- פעילויות ללא מיקום
- ימים ריקים
- טיסות חסרות

### סטטיסטיקות
- יעדים
- טיסות
- מלונות
- פעילויות
- מצב אריזה

### ניהול טיולים
- עריכת הטיול הפעיל
- שיתוף
- מעבר מהיר לטיולים אחרים
- ייבוא JSON
