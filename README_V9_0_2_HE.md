# Gal Family Trips – V9.0.2

## Targeted Listeners & Local Cache

### מאזינים ממוקדים
- מסך היום מאזין רק לפעילויות של היום.
- פירוט יום מאזין רק לפעילויות של היום שנבחר.
- טיסות מאזינות רק לאוסף flights.
- מלונות מאזינים רק לאוסף hotels.
- מסעדות מאזינות רק לאוסף restaurants.
- תקציב מאזין רק לאוסף expenses.
- מסמכים מאזינים רק לאוסף documents.
- ציוד מאזין רק לאוסף packing.

### ניהול חיבורים
- מעבר מסך סוגר את המאזין הקודם.
- החלפת טיול סוגרת את כל המאזינים הישנים.
- אין מאזין פעיל למסכים שאינם דורשים נתוני זמן אמת.
- עדכון שהגיע מהענן נשמר מקומית בלי להישלח חזרה כ-Diff.

### Cache
- Firestore persistence מופעלת במפורש.
- Cache מוגדר ללא הגבלת גודל.
- המסך מציג קודם את הנתונים המקומיים.
- עדכון הענן מחליף רק את האוסף הרלוונטי.
- תקלה ברשת אינה מנקה את הנתונים המוצגים.
