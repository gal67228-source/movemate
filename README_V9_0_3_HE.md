# Gal Family Trips – V9.0.3

## Conflict Protection

- לכל פעילות נשמרת גרסת הענן האחרונה.
- לפני עדכון מתבצעת Transaction מול Firestore.
- אם הגרסה בענן השתנתה, השינוי המקומי אינו דורס אותה.
- גרירה ושינוי שעות נבדקים יחד בתוך Transaction.
- Batch של סדר היום נשמר כולו או לא נשמר כלל.
- עדכון שהגיע מהענן כולל:
  - revision
  - updatedAt
  - updatedBy
- מטא-דאטה של הענן אינו נחשב לשינוי משתמש.
- במקרה של התנגשות מוצגת הודעה באפליקציה.
- השינוי המקומי נשאר שמור במכשיר.
- המשתמש יכול לטעון את העדכון מהענן או להשאיר
  את העותק המקומי להמשך טיפול.
