# Gal Family Trips – V6.3.1

## מזג אוויר אוטומטי לפי המסלול היומי

מיקום התחזית נקבע לכל יום בנפרד לפי סדר העדיפויות הבא:

1. יעד מהטיול הרב־יעדי שמופיע בפעילויות של היום.
2. כתובות ומיקומים של הפעילויות.
3. שמות אטרקציות ומקומות מרכזיים.
4. כותרת היום.
5. רשימת היעדים בטיול.
6. היעד הראשי של הטיול כברירת מחדל.

## תצוגה
- בכרטיס מזג האוויר מופיעים העיר והמדינה ששימשו לתחזית.
- שינוי מסלול או מיקום פעילות מרענן את שיוך התחזית.
- ימים בערים שונות מקבלים תחזיות נפרדות.
- הנתונים עדיין מוצגים רק בטווח התחזית הזמין.
- במצב אופליין לא מתבצע רענון.
