# Gal Family Trips – V7 Quick Route Builder

- לחיצה על + פותחת הוספה מהירה.
- סוגי פעילות מוכנים:
  - אטרקציה
  - ארוחה
  - מלון
  - טיסה
  - הסעה
  - רכבת
  - קניות
  - מנוחה
  - בריכה
  - טיול רגלי
- ממלאים רק שעה, שם, מיקום ומשך.
- משך, אמצעי הגעה והערות מקבלים ברירות מחדל.
- קישור Google Maps נוצר אוטומטית.
- ניתן לעבור לטופס המלא בכל עת.
- מוצעת שעה אוטומטית לפי הפעילות האחרונה ביום.
