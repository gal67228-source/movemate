# Gal Family Trips – V7.4

## גרירת פעילויות
- לכל פעילות נוספה ידית גרירה `☰`.
- לחיצה ארוכה וגרירה למעלה או למטה משנה את הסדר.
- הסדר נשמר מיד בתוך היום.
- רשימת הפעילויות אינה ממוינת יותר אוטומטית לפי שעה.
- אפשר עדיין לערוך את השעה בנפרד.
- פעילויות אוטומטיות וידניות ניתנות לגרירה באותה צורה.

## מפת LIVE
- נוסף כפתור מפת LIVE בראש היום.
- מוצגת הפעילות הבאה.
- מוצגות כל הפעילויות שנותרו.
- ניתן לפתוח ניווט לפעילות הבאה ב-Google Maps או Waze.
- ניתן לפתוח מסלול חי מהמיקום הנוכחי דרך כל התחנות שנותרו.
- לחיצה על "סיימתי את הפעילות" מסמנת אותה כבוצעה ומקדמת את הפעילות הבאה.
- פעילויות שכבר הושלמו אינן נכללות במסלול החי.

## ללא API נוסף
- המפה החיה נפתחת באפליקציית המפות המותקנת.
- אין צורך במפתח Google Maps חדש.
- כאשר אין origin בקישור, אפליקציית המפות משתמשת במיקום הנוכחי של המכשיר.
