# Gal Family Trips – V8.0.3

## שמירת מסלולים קבועה

- מסלול Google שנשמר אינו פג אוטומטית.
- פתיחת האפליקציה, מעבר בין מסכים או הפעלה מחדש אינם שולחים בקשה נוספת.
- כל מעבר מזוהה לפי חתימה הכוללת מוצא, יעד ואמצעי מעבר.
- בקשה חדשה נשלחת רק כאשר החתימה משתנה.
- שינוי סדר הפעילויות מחשב מחדש רק את המעברים שהשתנו.
- שינוי כתובת, קואורדינטות או אמצעי מעבר מחשב מחדש את המעבר המתאים.
- גם תוצאת הערכה לאחר כשל נשמרת כדי למנוע ניסיונות חוזרים בכל פתיחה.
- כפתור הרענון הידני מנקה את החתימות ומחשב את מעברי היום מחדש.
