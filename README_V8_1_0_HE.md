# Gal Family Trips – V8.1.0

## מסך "היום" החדש

- כרטיס פעילות נוכחית.
- כרטיס פעילות הבאה.
- זמן המעבר השמור בין הפעילויות.
- שעת יציאה מומלצת.
- מצב יציאה:
  - כדאי לצאת בעוד X דקות
  - צא עכשיו
  - איחור של X דקות
- כפתורי Google Maps ו-Waze ליד הפעילות הבאה.
- פס התקדמות יומי.
- מספר היום מתוך כלל ימי הטיול.
- סימון ברור בציר היום:
  - הושלמה
  - מתבצעת עכשיו
  - הפעילות הבאה
- מזג האוויר, השעון המקומי, המלון והטיסות נשארו במסך.
