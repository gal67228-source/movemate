# MoveMate 0.99.9

גרסת איחוד לקראת 1.0:

- הרשאות שיתוף: בעלים, עורך וצופה.
- קוד הזמנה יכול להעניק הרשאת עריכה או צפייה בלבד.
- בעל המעבר יכול לשנות הרשאה או להסיר משתמש.
- Viewer חסום מכתיבה גם מקומית וגם בכללי Firestore.
- ייצוא גיבוי מלא לקובץ JSON דרך חלון השיתוף של Android.
- ייבוא ושחזור גיבוי JSON מתוך מסך ההגדרות.
- Dashboard חכם עם משימות באיחור, פריטים ללא ארגז ומכירות ללא מחיר.
- בדיקות בסיס לפורמט הגיבוי ולתפקידי שיתוף.
