# MoveMate 0.95.0

This is the polish release before 1.0.

## Improvements

- Unified Material 3 styling across light and dark modes.
- Larger and more consistent touch targets.
- Consistent cards, dialogs, forms, buttons, chips, and list rows.
- Improved Android navigation transitions and predictive-back support.
- Added a theme regression test.
- CI now produces both an APK for testing and an AAB for Google Play preparation.
- Added a manual QA checklist covering upgrade safety and all core workflows.

## Scope

No new product feature was added. Existing local data formats and repositories remain unchanged.
