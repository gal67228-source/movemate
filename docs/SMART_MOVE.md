# Smart Move — v0.99.7

The Smart Move screen combines existing MoveMate data into one planning view.

- Readiness score: average of task, packing, box, shopping, and optional sales progress.
- Timeline: the next open tasks ordered by due date.
- Remaining inventory: moving items that are still marked as at home, grouped by room.
- Recommendations: missing standard tasks are added without duplicating existing task titles.

No background server or paid service is required for these calculations.
