# Gal Family Trips – V8.6.0.1

## קבלת SHA בלי Android Studio

נוסף ל-GitHub Actions שלב:
`Show Firebase SHA fingerprints`

בכל הפעלת Build הלוג יציג:
- SHA1
- SHA256

### איך למצוא את הערכים

1. העלה את הפרויקט ל-GitHub.
2. פתח את Actions.
3. הפעל את Workflow הבנייה.
4. פתח את השלב `Show Firebase SHA fingerprints`.
5. העתק את SHA1 ואת SHA256.
6. הוסף אותם ב-Firebase:
   Project Settings → Your apps → Add fingerprint.
