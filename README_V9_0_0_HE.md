# Gal Family Trips – V9.0.0

## Cloud Data Model

גרסה זו מניחה תשתית לסנכרון Diff.

### מבנה Firestore החדש
- מסמך טיול בסיסי בלבד.
- תתי-אוספים נפרדים לימים, פעילויות, טיסות,
  מלונות, מסעדות, תקציב, ציוד ומסמכים.

### יעילות
- עדכון פעילות שולח רק שדות שהשתנו.
- גרירה משתמשת ב-Batch ורק בפריטים שהשתנו.
- מחיקה מוחקת מסמך יחיד.
- כתיבה ראשונית מחולקת לקבוצות של עד 400 מסמכים.
- ימים נשמרים ללא רשימת הפעילויות הכבדה.

### מעבר מדורג
היישום הישן נשאר זמני לצורך תאימות לטיולים קיימים.
ב-V9.0.1 נחבר את מסכי הפעילויות ל-Diff Sync
ולתור שינויים מקומי.
